# NanoTS

A high-performance TSDB prototype designed for extreme memory efficiency.

### Key Features
- **Gorilla Compression:** Implements XOR-based float compression and Delta-of-Delta timestamp encoding.
- **Cache-Optimized:** Custom memory layout designed to minimize cache misses during ingestion.
- **Zero-Dependency:** Pure C++ implementation to maximize portability and performance.

*Under active development.*
